# Governance & Compliance

Document regulatory requirements, privacy impact assessments, responsible AI policies, and cross-border data sharing agreements here.

## Templates
- `privacy/` – DPIA templates, consent models, and data retention policies.
- `responsible-ai.md` – guardrails, audit procedures, and transparency reports.
- `federation-agreements/` – memoranda of understanding and API usage terms.

Keep this directory synchronized with legal and policy reviews.
